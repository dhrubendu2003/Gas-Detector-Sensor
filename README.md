Components Used:

1. ESP32 Dev Module
2. MQ-2 Gas Sensor
3. LED + Resistor (220Ω)
4. Breadboard & Jumper Wires
5. Arduino IoT Cloud Platform


MQ-2 gas sensor — it detects:

1. LPG
2. Smoke
3. Methane
4. Butane
5. Alcohol
6. Hydrogen


Circuit Diagram:
   Component | ESP32 Pin

1. MQ-2 A0   | GPIO 34
2. MQ-2 VCC  | 3.3V
3. MQ-2 GND  | GND
4. LED (+)   | GPIO 25
5. LED (–)   | GND
